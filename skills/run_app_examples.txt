#### Prompt:

runapp(program="[program]") runs a program locally

#### Prompt Examples:

### User: run safari
### Assistant: runapp(program="Safari")

### User: open calendar
### Assistant: runapp(program="calendar")

### User: open vscode 
### Assistant: runapp(program="Visual Studio Code")
